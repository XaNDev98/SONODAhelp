ERRO UNC: O que fazer se ocorrer mensagem de erro "Compartilhamento de rede UNC" ?

Este erro compartilhamento de rede UNC pode ocorrer por algumas razões como: compartilhamento mapeamento errado da pasta ou o usuário não possui senha para acessar arquivos ou pastas compartilhadas.

Solução: Verifique se o Compartilhamento protegido por senha está ativo.

PROCEDIMENTOS SERVIDOR/ATALHO:

1- Desabilitar o Anti-Vírus, Firewall e Windows Defender;
2- Na Pasta do sistema você deve compartilhar o ponto em rede;
3- Você irá ir em: PAINEL DE CONTROLE / CENTRAL DE REDE E COMPARTILHAMENTO / ALTERAR AS CONFIGURAÇÕES DE COMPARTILHAMENTO AVANÇADAS;
4- Abra a aba TODAS AS REDES e em COMPARTILHAMENTO PROTEGIDO POR SENHA deixe a opção DESATIVAR COMPARTILHAMENTO PROTEGIDO POR SENHA.


ERRO COLETA NÃO FUNCIONA: O que fazer se ocorrer mensagem de erro "Coleta automática funciona" ?

Este erro Coleta automática não funciona pode ocorrer quando o Agente de Comunicação não executável.

Solução: faça o passo a passo abaixo:

PROCEDIMENTOS:

1- Com o sistema aberto, vá em MANUTENÇÃO > EQUIPAMENTOS > AGENDAR COMUNICAÇÃO, e apague a coleta, depois feche o sistema;
2- Acesse a pasta do ponto, localize e execute o REMGER.EXE como ADMINISTRADOR;
3- Procure se tem o arquivo LOG-ERROS.TXT, se tiver, apague este arquivo;
4- Localize e execute o EXATP4.EXE como ADMINISTRADOR;
5- Procure o arquivo PONTO4DLL.DLL, segure a tecla SHIFT e clique com o botão direito do mouse, selecione a opção COPIAR COMO CAMINHO ou COPY AS PATH caso esteja em inglês;
6- No windows abra o CMD como ADMINISTRADOR e digite regsvr32 e depois cole o caminho do arquivo que você acabou de copiar do PONTO4DLL.DLL;
7- Ainda na pasta do ponto, localize e execute o INSTGER.EXE como ADMINISTRADOR;
8- Inicie o gerenciador de tarefas do sistema e olhe na pasta se gerou um novo arquivo LOG-ERROS.TXT;
9- Abra o sistema e crie uma nova coleta automática.


ERRO 183: O que fazer se ocorrer mensagem de erro "código 183" ?

Este erro Código 183 ocorre quando é feito a tentativa de atualização do sistema no servidor, e o Ponto está sendo utilizado por outra aplicação ou por outro usuário da rede.

Solução: faça o passo a passo abaixo:

PROCEDIMENTOS:

1- No menu iniciar, pesquise por GERENCIMENO DO COMPUTADOR;
2- Vá em PASTAS COMPARTILHADAS > SESSÕES e clique com o botão direito do mouse e clique em DESCONECTAR TODAS AS SESSÕES.


INSTALAR MAIS DE UM GERENCIADOR DE TAREFAS

Solução: faça o passo a passo abaixo:

1- BAIXE O INSTALADOR AlterarInstancia.exe;
2- COLOQUE O AlterarInstancia.exe NA PASTA DO SISTEMA;
3- EXECUTE O AlterarInstancia.exe COMO ADMINISTRADOR;
4- PREENCHA COM O BANCO DE DADOS E DEPOIS CLIQUE NO BOTÃO ALTERAR;
5- PREENCHER O NOME DO SERVIÇO E CONCLUIR.


ERRO 482: O que fazer se ocorrer mensagem de erro "Run-time error: 482: Printer error" ?

Solução: faça o passo a passo abaixo:

1- APERTE O BOTÃO DO WINDOWS E PESQUISE PAINEL DE CONTROLE > PROGRAMAS E RECURSOS
2- VÁ EM ATIVAR OU DESATIVAR RECURSOS DO WINDOWS
3- CLIQUE NA CAIXINHA E ATIVE O IMPRESSÃO DA MICROSOFT EM PDF


ERRO 52: O que fazer se ocorrer mensagem de erro "52: bad file name or number"?

A mensagem Bad file name or number ocorre quando não há permissão para acesso completo de um usuário a um determinado arquivo (normalmente o banco de dados).

Solução: Certifique-se de que as permissões de leitura e escrita estão corretas para o arquivo de dados do sistema.

1- GERALMENTE ESTÁ EM C:\Program Files\Microsoft SQLServer\MSSQL14.SQL2017\MSSQL\Backup
2- VERIFICAR SE O LOCAL ONDE É GERADO O BACKUP ESTÁ CORRETO
3- INSERIR PERMISSÃO NA PASTA DA INSTÂNCIA DO BANCO DE DADOS


TICKET PENDENTE

A mensagem Ticket Pendente ocorre quando tem algum papel enroscado na impressora do relógio.

Solução: Certifique-se que a bobina esteja colocada de forma correta e se não tem nenhum papel enroscado na impressora e depois faça o passo a passo abaixo:

1- FAZER O DOWNLOAD DO PROGRAMA repidclass_config.exe E DEPOIS CRIE UMA PASTA E COLOQUE O PROGRAMA DENTRO.
2- EXECUTAR O ARQUIVO repidclass_config.exe PELO CMD NA PASTA DO ARQUIVO ONDE FOI BAIXADO
3- APERTAR O BOTÃO TAB ATÉ APARECER O ARQUIVO BAIXADO
4- APÓS ENCONTRAR O ARQUIVO, DIGITE NA FRENTE DO NOME DO ARQUIVO /dev E APERTE A TECLA ENTER
5- COLOQUE O IP DO RELÓGIO > LER CONFIGURAÇÕES > SEM SENSOR, QUANDO APARECER SENSOR 0 REINICIE O RELÓGIO DE PONTO.
